Prefixes

Use any of these prefixes to run commands:

? ! % $ .

Server admins can set a custom prefix using:

?config prefix <new_prefix>

Moderation commands

ban

Requires: Ban Members
Aliases: kill

Ban a user from the server (works on users not in the server)

?ban <user> [reason]

Example:

?ban @user Breaking rules

tempban

Requires: Ban Members

Temporarily ban a member

?tempban <user> <duration> [days] [reason]

Example:

?tempban @user 7d Timeout for a week

softban

Requires: Ban Members

Ban and immediately unban to clear message history

?softban <user> [days=2] [reason]

massban

Requires: Ban Members

Ban multiple members at once

?massban [days=2] <users...>

warns

Requires: Kick Members

Warning management system

?warns [user]                  # View warnings
?warns remove <case_id>       # Remove a warning
?warns clear <user>           # Clear all warnings

warn

Requires: Kick Members

Warn a member

?warn <user> [reason]

purge

Requires: Manage Messages

Delete multiple messages

?purge [amount=100] [user]    # Delete messages
?purge bot [amount] [prefix]  # Delete bot messages
?purge contains [amount] [text] # Delete messages containing text

lock

Requires: Manage Channels

Lock a channel

?lock [channel]

unlock

Requires: Manage Channels

Unlock a channel

?unlock [channel]

nickname

Requires: Manage Nicknames
Aliases: nick

Change a member's nickname

?nickname <user> [new_nickname]

Utility commands

afk

Set your AFK status

?afk [message]

reminder

Aliases: remind, remindme

Set a reminder

?reminder <time> <message>

Examples:

?reminder 2h30m check the oven ?reminder 1d daily reminder ?reminder 30m quick reminder

reminders

List your active reminders

?reminders

tag

Requires: Manage Messages

Custom server tags/commands

?tag create <name> <content>  # Create tag
?tag remove <name>          # Remove tag
?tag list                    # List all tags
?tag <name>                 # Use a tag

steam

Display Steam profile information

?steam <steamID/customURL/profileURL>

Fun

roll

Roll dice in NdN format

?roll [dice=1d6]

Examples:

?roll 2d6 ?roll 1d20 ?roll 4d4

rps

Play Rock Paper Scissors

?rps <rock|paper|scissors>

uwu

UwU-ify text

?uwu <text>

emojify

Convert text to regional indicators

?emojify <text>

hug

Hug someone

?hug <user> [intensity=1]

bean

Aliases: punish, troll, bange, jail

Bean a user (fake ban command)

?bean <user>

cookie

Cookie system commands

?cookies [user]           # Check cookies
?give <user> [amount]     # Give cookies
?eat [amount]             # Eat cookies

8ball

Ask the magic 8ball

?8ball <question>

choose

Choose between options

?choose option1, option2, option3...

pat

Pat someone

?pat <user>

bonk

Bonk someone

?bonk <user>

slap

Slap someone with a random object

?slap <user>

Information commands

profile

Aliases: userinfo, user

Get detailed info about a user

?profile [user]

avatar

Aliases: av, avi, pfp

Get a user's avatar

?avatar [user]

steam

Display a Steam profile

?steam <steamID/customURL/profileURL>

Administration commands

config

Requires: Administrator

Configure server settings

?config prefix <prefix>     # Set custom prefix
?config messages <channel>  # Set message logs
?config mod_audit <channel> # Set mod logs
?config join_leave <channel> # Set join/leave logs

toggleprefix

Requires: Administrator
Aliases: tp

Toggle whether default prefixes work

?toggleprefix

starboard

Requires: Administrator
Aliases: sb

Manage starboard settings

?starboard <channel>      # Set channel
?starboard limit <number>  # Set minimum stars
?starboard view           # View settings
?starboard disable       # Disable starboard

Integration commands

twitch

Requires: Manage Guild

Manage Twitch notifications

?twitch channel <channel>     # Set notification channel
?twitch add <streamer> [role] # Add streamer
?twitch remove <streamer>     # Remove streamer
?twitch role <role>          # Set global ping role

youtube

Requires: Manage Guild

Manage YouTube notifications

?youtube channel <channel>   # Set notification channel
?youtube add <channel> [role] # Add YouTube channel
?youtube remove <channel>     # Remove channel
?youtube role <role>         # Set global ping role
