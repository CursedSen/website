In-depth documentation on how to use the bot with examples coming soon!

For now, here are the available features:

~/Documentation
  /Core Features
    /Moderation
      - User Management (kick, ban, timeout, mute/unmute)
      - Record Keeping (audit logs, user notes, warnings)
      - Message Control (purge, edit/delete logging)
      - Server Management (join/leave logs, profile logs)

    /Levels
      - Message-based XP gain
      - Configurable level roles
      - Leaderboard tracking
      - Level-up notifications
      - Message count tracking

    /Economy
      - Virtual currency system
      - Item shop with usable items
      - Gambling with buckshot roulette
      - Betting system with stats

    /Fun Commands
      /Text Manipulation
        - mock: SpOnGeBoB text
        - mamamia: Italianize text
        - uwu: uwu speak
        - bean: Fake punishment
      /Games
        - Buckshot roulette
        - Coinflip
        - Dice rolling

  /Configuration
    /Custom prefix per server
    /Configurable log channels
      - Join/Leave events
      - Moderation actions
      - Message edits/deletions
      - Profile changes
    - Starboard system
      - Custom threshold
      - Multiple reaction types

  /Administrative
    - Bot masters only:
      - blacklist: Block users from using bot
      - update: Pull latest changes
      - restart: Restart the bot
    - Server admins:
      - config: Configure server settings
      - indexmessages: Bulk process messages for level indexing
      - makeroles: Create level roles

  /Additional Features
    - Link fixing (Twitter/Reddit)
    - User profile tracking
    - Message quoting system
    - Comprehensive error handling
    - SQLite database storage
    - Role-based permissions

For detailed command usage, use -help or -help [command]
