~/Features

    - Message cleanup and filtering
    - Auto-moderation capabilities
    - User event logging
    
  /Fun
    - Mini-games *WIP*
    - Image manipulation
    - Text manipulation
    - Leveling system

~/Usage
  /Prefix:
    All commands start with the hyphen prefix (-)
    This is configurable
    
  /Common Commands
    help          - Shows command list
    info          - Bot information
    ping          - Check bot latency
    server        - Server details
    
  /Moderation
    kick          - Kick a user
    ban           - Ban a user
    mute          - Timeout a user
    purge *num*   - Delete messages

~/Setup
  1. Invite Rei to your server using this link
  
  2. Ensure Rei has proper permissions
  
  3. Use "-help" to get started
